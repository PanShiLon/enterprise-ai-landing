# 企业AI落地实战：从RAG知识库到ReAct智能体

本仓库是掘金小册《企业AI落地实战》的配套代码。

> 小册购买链接：即将上线

## 环境要求

- Docker 24.0+
- Python 3.10
- Java 17（第八章 Agent 代码）

## 快速启动

```bash
git clone https://github.com/PanShiLon/enterprise-ai-landing.git
cd enterprise-ai-landing
docker compose up -d
```

## 代码目录

| 章节 | 内容 | 路径 |
